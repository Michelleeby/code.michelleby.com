ABOUT

What is this site? Besides a sandbox to store solutions to algorithm problems, this is a static website powered by a spreadsheet.

First, the content of this website was designed and organized using relational database principles. Then, it was translated into a Google spreadsheet. Although a spreadsheet is different from a database, when the data is organized using the same principles, it can be interfaced with in similar ways.

Jekyll is the static site generator used to build this website. Out of the box, Jekyll supports working with .csv data files. To interface with the spreadsheet, first a generator plugin calls a Google Sheets API service, it then pulls the data, and finally it writes a .csv file for each fetched sheet. After data pages have been created, Jekyll builds the site as usual, with the addition of a few extra filters.

After Jekyll generates the site files, they are pushed to the gh-pages branch of this site’s repository. The main branch of this site’s repository is what Jekyll uses when it builds locally. The gh-pages branch houses the static files generated by Jekyll that GitHub hosts.

Finally, once the files are pushed to the gh-pages branch, this site is live!

PROMPTS

TWO SUM

Given an array of integers and a target integer, return the indices of the two numbers that total the target when summed. Assume each input has exactly one solution, and that each element is used only once.

INPUT

ints = [2, 7, 9, 15],
target = 11

OUTPUT

[0, 2]

MAX DIFFERENCE

Given an array of integers, find and return the maximum difference between the ith number and another number that follows.

INPUT

ints = [15, 3, 7, 1, 6, 8]

OUTPUT

7

COUNT ISLANDS

Given a 2D binary matrix where '1' is land and '0' is water, return the total number of islands. An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically. You may assume all four edges of the grid are all surrounded by water.

INPUT

matrix = [
["1", "1", "1", "0", "1"],
["1", "1", "0", "0", "0"],
["1", "1", "1", "0", "0"]
]

OUTPUT

2

SUBSEQUENCES

Given two strings s and t, return the number of distinct subsequences of s which equals t.

INPUT

s = "legeegg", t = "leg"

OUTPUT

7

SUBSETS

Given an integer array of unique elements, return all possible unique subsets as an array. Return the solution in any order.

INPUT

ints = [0, 1, 2]

OUTPUT

[
[0, 1, 2],
[0, 1],
[0, 2],
[0],
[1, 2],
[1],
[2],
[]
]

K CLOSEST

INPUT

points = [
[1, 0],
[2, 0],
[3, 0]
],
k = 1

OUTPUT

[
[1, 0]
